# 2025년 새싹 해커톤 AI 서비스 기획서

## 팀 정보
- **팀명**: 동에번쩍
- **팀 구성원**: 5명 (개발 3명, 기획 2명)

## 1. AI 서비스 명칭

**BOBSCAN (밥스캔)** - 초개인화 AI 메뉴 추천 서비스

---
## 2. 활용 AI 기술 및 API

| 번호 | AI 기술 / API | 출처 / 용도 |
|------|---------------|-------------|
| 1 | Google Cloud Vision API | GCP / 메뉴판 텍스트 실시간 인식 |
| 2 | Gemini 2.0 Flash | Google AI / 번역·분석·개인화 추천 |
| 3 | Google Cloud Translation API | GCP / 보조 번역 (필요 시) |

※ **학습용 데이터 불필요** - 사용자 스캔 + 취향 데이터를 AI가 실시간 조합

---
## 3. 핵심내용

**초개인화 AI가 메뉴판과 사용자 취향을 실시간으로 분석하여 맞춤 추천**

사용자가 설정한 식습관(비건/알레르기/고기 선호/매운맛)과 메뉴판 데이터를 AI가 즉시 조합.
단순 번역을 넘어 "이 메뉴는 당신에게 추천/주의/비추천" 판단 제공.
한국 음식 원어(Bibimbap) + 로마자 발음으로 문화적 경험까지 전달.

---
## 4. 제안배경 및 목적

### ◆ 제안 배경
- 외국인 관광객 1,400만명 시대, 메뉴 이해 실패로 주문 포기 빈번
- 기존 번역 앱: 단순 텍스트 변환, 개인 식습관 고려 전무
- 비건/알레르기 정보 확인 어려움으로 식사 선택 제한

### ◆ 목적
- 개인 맞춤 추천으로 안전하고 만족스러운 식사 경험 제공
- 한국 음식 원어 표기로 문화 원형 보존 및 확산
- 언어 장벽 없는 한식 경험으로 관광 만족도 향상

---
## 5. 기대효과

### ◆ 사회·문화적 효과
- 개인 맞춤 추천으로 안전한 식사 선택 (알레르기/식습관 보호)
- 한국 음식 원어 표기로 문화 원형 보존 및 글로벌 확산
- 외국인이 "Bibimbap"을 직접 발음하며 한식 문화 체험

### ◆ 경제적 효과
- 메뉴 이해도 향상으로 주문 만족도 증가
- 언어 장벽 완화로 외국인 재방문율 상승 기대
- 한식 브랜드 가치 상승 (K-Food 문화 확산)

---
## 6. 세부내용

### ◆ AI 활용 방식

- **Vision API**: 메뉴판 사진에서 텍스트 + 위치 정보 실시간 추출
- **Gemini 2.0 Flash**: 번역 + 재료 분석 + 개인화 추천을 1회 호출로 처리
- **사용자 취향 프리셋(로컬 저장) + 메뉴 데이터 → AI가 실시간 매칭**

---
### ◆ 서비스 플로우
```
[Step 1] 취향 설정
         비건/알레르기/고기 선호/매운맛 등급 선택 (로컬 저장)

[Step 2] 메뉴판 촬영
         카메라 or 갤러리에서 메뉴판 스캔

[Step 3] 개인화 추천
         AI가 각 메뉴를 분석하여 추천도 표시 (추천/주의/비추천)

[Step 4] 주문 지원
         선택 메뉴 → 환율 변환 → 음성 주문 문장 생성
```

---
### ◆ 핵심 차별점
1. **초개인화 추천**: 사용자 취향 × 메뉴 데이터 실시간 조합으로 맞춤 판단
2. **문화 보존**: "Mixed Rice"가 아닌 "Bibimbap(비빔밥)" 원어 + 발음 제공
3. **속도 최적화**: AI 1회 호출로 번역+분석 동시 처리 (2초 이내)

---
### ◆ 기술 구조
- **Frontend**: Streamlit (프로토타입)
- **Backend**: Python FastAPI (GCP Cloud Run 배포)
- **AI Pipeline**: 이미지 → OCR → Gemini(번역+분석+추천) → JSON 응답
- **배포**: Google Cloud Platform

---
### ◆ 주요 기능
1. **취향 프리셋**: 비건/알레르기/고기/매운맛 설정, 로컬 저장
2. **개인화 추천**: 각 메뉴마다 추천/주의/비추천 + 이유 제시
3. **원어 표기**: Bibimbap(비빔밥) 형식으로 문화 전달
4. **환율 변환**: 메뉴 가격을 사용자 통화로 표시
5. **음성 주문**: TTS로 한국어 주문 문장 읽어주기

---
### ◆ 창의성 및 구현 가능성
- **창의성**: 단순 번역이 아닌 "초개인화 AI 추천" + "문화적 경험"까지 제공
- **구현 가능성**: GCP API 활용으로 해커톤 기간 내 완성 가능
- **확장성**: 음성 인식 주문, 리뷰 요약, 맛집 추천 추가 가능
